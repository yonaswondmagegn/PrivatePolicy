Privacy Policy

This Chrome extension respects your privacy. This document explains what data we collect, why we collect it, and how it is used.

1. History

We use browsing history only to improve the functionality of the extension, such as determining the user’s previous navigation path or restoring state.

2. Identity

We do not collect personally identifiable information unless explicitly required for authentication with third-party services. In such cases, the data is used solely for session identification and is not stored or shared.

3. Remote Code

The extension may load remote scripts for update or feature delivery purposes. These scripts are secure, hosted on trusted servers, and are strictly necessary for the extension's functionality.

4. Sessions

Sessions are used to maintain the state of the extension while it's running. This helps in providing a seamless user experience.

5. Storage

We use Chrome’s `storage` API to store preferences, settings, or temporary data locally. None of this data is transmitted or shared.

6. Tabs

The extension may access tab data to perform its core functions, such as interacting with content on the page or injecting scripts.

7. Web Navigation

Web navigation data is only used to detect when certain pages are loaded, which enables features of the extension relevant to those pages.

8. Data Sharing

We do not sell, rent, or share your data with any third parties.
